# openclaw_command_notebook
openclaw相关笔记
- 重新部署openclaw
```bash
openclaw onboard
```

- 重新配置某一项
```bash
openclaw configure
```

- 重新启动openclaw
 ```bash
openclaw gateway restart
``` 

- 给openclaw安装skills(举例安装gemini)
 ```bash
npx clawhub@latest install gemini
```

-卸载openclaw
```bash
npm uninstall -g clawdbot moltbot openclaw
```

-安装最新版的openclaw
```bash
npm install -g openclaw@latest
```
